IFC Merge

Unisce più file IFC in un unico modello federato: un solo IfcProject, un IfcSite dedicato e riconoscibile per ogni file. Geometrie, property set, tipi, materiali e colori restano al loro posto.

100% nel tuo browser — nessun file viene caricato su un server.

1File IFC da unire (l'ordine conta: il primo fa da base)

Il primo file mantiene intera la sua struttura spaziale (Site/Building/Storey) e presta a tutti il proprio IfcProject; ogni file successivo viene ricollegato sotto un nuovo IfcSite col nome del file. Trascina qui i file o usa i pulsanti per riordinarli.

Nessun file selezionato — servono almeno due IFC.

2Opzioni

Nome IfcProject
Nome file di output

I file dovrebbero condividere schema e unità di lunghezza (es. tutti in metri): in caso contrario il log lo segnala. Sistemi e zone (IfcRelAssignsToGroup) non vengono ricreati nel federato.

3Riepilogo per modello

# File IfcSite Elementi Note
Il riepilogo appare dopo l'unione.
